# Python Bootcamp

Shared Python foundations for all MSCF summer programs. This course is taken by students across the ML Post-Bacc, Summer Bridge Experience (SBX), and AI Animation programs.

## Programs Using This Curriculum

- [ML Post-Bacc](https://github.com/morehouse-supercomputing/ml-postbacc)
- [SBX Python](https://github.com/morehouse-supercomputing/sbx-python)
- AI Animation (external program)

## Structure

```
python-bootcamp/
  lessons/        -- daily lesson materials
  notebooks/      -- Jupyter notebooks and exercises
  assignments/    -- homework and projects
  resources/      -- reference sheets, setup guides
```
