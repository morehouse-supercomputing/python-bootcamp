# Python Bootcamp

Shared Python foundations for all MSCF summer programs. This 3-week course is taken by students across the ML Post-Bacc, Summer Bridge Experience (SBX), and other summer programs.

**Course site:** [morehouse-supercomputing.github.io/python-bootcamp](https://morehouse-supercomputing.github.io/python-bootcamp/)

## For Students

1. Clone this repo:
   ```bash
   git clone https://github.com/morehouse-supercomputing/python-bootcamp.git
   ```
2. Before each lesson, pull the latest:
   ```bash
   cd python-bootcamp
   git pull
   ```
3. Open the notebook for the day's lesson, complete the exercises
4. Submit your completed notebook to Canvas
